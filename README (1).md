# Élégance - Apparel Showroom

A modern, responsive apparel showroom website with PWA capabilities.

## Files Included

- `index.html` - Main HTML file
- `styles.css` - Styling
- `app.js` - JavaScript functionality
- `manifest.json` - PWA manifest
- `service-worker.js` - Service worker for offline functionality
- `icon-192.png` - App icon (192x192)
- `icon-512.png` - App icon (512x512)

## How to Deploy on Netlify

### Method 1: Drag and Drop (Easiest)
1. Go to https://app.netlify.com/
2. Drag your entire project folder to the deploy area
3. Done! Your site will be live in seconds

### Method 2: Git Deploy
1. Push your code to GitHub
2. In Netlify, click "New site from Git"
3. Connect your repository
4. Build settings:
   - **Build command:** (leave empty)
   - **Publish directory:** `.` or leave empty
5. Click "Deploy site"

## Troubleshooting

If you see "Page not found":
- Check that `index.html` is in the root directory
- Verify Publish directory is set to `.` or empty
- Make sure all files are uploaded (not shortcuts)
- Clear cache and redeploy

## Local Development

Simply open `index.html` in your browser to view locally.
